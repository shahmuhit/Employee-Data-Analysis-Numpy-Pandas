# Employee Data Analysis using NumPy and Pandas

## Project Overview

This project demonstrates the use of NumPy and Pandas for employee dataset generation and analysis. A synthetic employee dataset is created and various data analysis operations are performed, including statistical calculations, filtering, sorting, conditional updates, and role-based salary analysis.

## Features

* Generate a synthetic employee dataset
* Perform statistical analysis on employee records
* Calculate maximum, minimum, and average salary
* Calculate average employee age
* Filter employees based on age and salary criteria
* Apply conditional updates to employee records
* Simulate salary increments for specific employee roles
* Load and analyze CSV data using Pandas
* Sort and filter employee information
* Extract department-specific employee data

## Technologies Used

* Python
* NumPy
* Pandas
* Jupyter Notebook

## Dataset Attributes

* Employee ID
* Employee Age
* Employee Salary
* Employee Role

## Project Tasks

1. Employee dataset generation
2. Statistical computations
3. Data filtering and selection
4. Conditional record modification
5. Salary analysis
6. DataFrame operations
7. Employee sorting
8. Department-based data extraction
